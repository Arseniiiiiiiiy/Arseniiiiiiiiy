## Проекты

### Price Monitor Bot
Учебный backend-сервис для мониторинга цен товаров через API
с сохранением истории и уведомлениями в Telegram.

 https://github.com/Arseniiiiiiiiy/price-monitor-bot

Проект находится в активной разработке.

В проекте реализовано:
REST API на FastAPI;
- асинхронные эндпоинты;
- хранение истории цен в SQL-базе;
- расчёт дельты цены без дублирования данных;
- healthcheck и graceful shutdown;
- централизованное логирование;
- Telegram-бот для взаимодействия с сервисом.
